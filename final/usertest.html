Final Project User Test Session & Results

Observations From User Testing

  1. Users tend to have a hard time noticing which section is clickable

    2. The only way that indicates a user how a section is clickable is by the hover change. I found that all three user testers seemed lost in finding that section especially since I only had one section prepared. Some suggestions I received regarding to this problem are animating to a bigger image when hover or adding filter or color that fades in and out.

  2. Users like the minimalistic style

    3. Designing in a minimalistic style and focusing on the images and story is a good way for the users to have a more immersive experience. Users like the simple close button that has a x icon because adding a geometric shape like a round button would not match with the rest of torn paper like style.

  3. Adding more interactions and descriptions is recommended

    4. Adding animations would not only make it more interesting to the users, but also gives a hint what they should do. Writing a short description about the overall project and each image would tell a better story that is more personal to me.
